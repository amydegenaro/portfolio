Parkitech

Parkitech is a geogtagged ticket board for park management that was created to fill a technology gap often found in small non-profits managing public parks and spaces. Maintenance and Park Ranger teams work both in the field and the office and need a good way to track and record tasks or issues from both places at once. Parkitech is a ticket board with a responsive interface for use on mobile, tablet, and desktop devices. Tickets are geotagged to record and plot their exact coordinates on a map, allowing for better tracking, organization, and analytics of park operations.

Built With

Node · Express · PostgreSQL · Sequelize · React · Redux · Mapbox

  • Website
  • GitHub

SpellBinder

An e-commerce web application that sells a variety of magic spells. SpellBinder showcases a dynamic shopping cart component with update and delete functionality. Shoppers can search for spells by name, filter by category or type, and add one or more spells to their cart. Shoppers can add to their cart without an account, but must login to checkout with Stripe. The contents of a shopper's cart is carried over and saved once the shopper logs in.

Built With

Node · Express · PostgreSQL · Sequelize · React · Redux · Stripe · Bootstrap
